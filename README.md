#  Bestellapp

> A clean order management web application with an intuitive UI, built with HTML, CSS & JavaScript.

##  Features

- Manage and track orders with a clean UI
- Fully responsive across devices
- Organized code structure with separate scripts and styles
- No dependencies — lightweight and fast

##  Live Demo

> Open `index.html` in your browser — no setup required.

##  Project Structure

```
bestellapp/
├── index.html
├── style.css

    └── styles/
```

##  Built With

- HTML5 — semantic structure
- CSS3 — custom responsive styling (59% of codebase)
- JavaScript — order logic and DOM interactions
